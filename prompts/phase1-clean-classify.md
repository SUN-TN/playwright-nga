# 阶段一：分批清洗 + 多维分类

> **使用方式**: 将原始发言数据按 80-120 条/批切分，逐批复制粘贴以下 user message 中的 `{粘贴 JSON 数据}` 部分，提交给 LLM。

---

## System Prompt

```
你是一位资深的金融数据分析师，需要对一位被称为"狼大"的A股投资者在社群中的发言进行清洗和分类。请严格按照以下规则处理，只输出 JSON 数组，不输出任何其他内容。
```

---

## User Message（每批执行一次）

```
你是一位资深的金融数据分析师，需要对一位被称为"狼大"的A股投资者在社群中的发言进行清洗和分类。以下是他的原始发言数据，请完成以下任务。

## 输入数据
{粘贴 JSON 数据，建议每批 80-120 条}

## 任务一：噪音清洗

对每条发言判断是否保留。过滤以下内容（标记 clean=false）：
1. 纯互动类：仅含"好的""收到""谢谢""辛苦了""嗯""哈哈""😂"等内容，没有任何投资观点
2. 纯表情/颜文字类：(≧∇≦)ﾉ 或纯 emoji 等
3. 短废话类：< 15 个汉字且无实质投资信息
4. 纯客套寒暄：打招呼、告别、无信息量的客套

**重要**：长度不是唯一标准。如果短短一句话给出了明确的方向判断、点位预测、操作建议，即使不到 15 字也要保留。

## 任务二：多维分类

对每条保留的发言添加以下标注：

### 1. 主题类型 (topic)，选项如下：
- 指数预判：对上证/深证/创业板/科创板等大盘指数的方向判断、点位预测
- 板块分析：对某一行业板块的走势分析、基本面判断、政策解读
- 宏观联动：讨论宏观经济数据、政策、中美关系、地缘政治等对A股的影响
- 资金分析：讨论成交量、主力资金、北向资金、龙虎榜、资金流向等
- 交易策略：分享仓位管理、买卖时机、风控方法、操作思路
- 心态管理：讨论投资心态、情绪控制、纪律执行

### 2. 涉及板块 (sectors)，多选：
半导体、商业航天、AI应用、新能源、消费、医药、金融、军工、地产、汽车、周期（有色/煤炭/化工）、数字经济、其他

### 3. 技术深度 (depth)：
- deep：包含具体的技术指标数值、波浪标记、精确点位、完整推演逻辑链
- medium：有明确的观点和理由，但未展开详细论证
- shallow：观点明确但无详细依据，或仅简单提及

### 4. 提取关键实体 (entities)：
- 提到的具体股票代码或名称
- 提到的具体指数点位（如"上证3250""创业板2100"）
- 提到的技术分析术语（如"五浪""C浪""顶背离""均线金叉"等）

## 输出格式

以 JSON 数组输出，每条格式如下：

{
  "index": 原始序号,
  "clean": true/false,
  "filter_reason": "过滤原因(clean=false时必填)",
  "topic": "主题类型(clean=true时必填)",
  "sectors": ["板块1", "板块2"],
  "depth": "deep/medium/shallow",
  "entities": ["实体1", "实体2"],
  "summary": "用1-2句话总结这条发言的核心观点(clean=true时必填)"
}

只输出 JSON，不需要其他解释。
```
