# 阶段二：知识抽取

> **前置条件**: 已完成阶段一所有批次的清洗分类，并合并为一个完整的 JSON 数组。
>
> **使用方式**: 将所有有效发言（clean=true）按日期排序后，将合并结果粘贴到 `{粘贴阶段一的合并结果}` 处提交给 LLM。
>
> **注意**: 如果数据量太大超出上下文限制，可按月或按周分批，用同一条提示词分别抽取后再手动合并。

---

## System Prompt

```
你是一位资深的技术分析师，擅长艾略特波浪理论、技术指标分析和A股市场板块轮动研究。请从发言数据中系统性提取知识，只输出结构化 JSON，不输出任何其他内容。
```

## User Message

```
你是一位资深的技术分析师，以下是一位被称为"狼大"的A股投资者在 2026年1月至6月期间的全部有效发言（共{N}条，已按日期排序并附分类标注）。请基于这些发言进行系统性的知识抽取。

## 输入数据
{粘贴阶段一的合并结果，如果太大可分批给，然后用同一条提示词分别抽取}

## 任务：系统性知识抽取

### 一、波浪理论标记体系
提取所有涉及艾略特波浪理论的发言，整理出：
1. 月线级别波浪标记（大周期定位）及变化时间线
2. 周线级别波浪标记（中周期定位）及变化时间线
3. 日线级别波浪标记（短周期操作指引）
4. 各周期波浪之间的嵌套关系（如"周线处于五浪中的三浪"）
5. 波浪标记发生过修正或变化的节点及原因
6. 对于当前所处波浪位置的最终判断

请按时间顺序组织，标注每次标记出现的日期。

### 二、关键点位数据库
提取所有具体点位判断，整理成表格：
| 日期 | 指数（上证/创业板/科创等） | 点位 | 性质（支撑/压力/目标） | 判断依据 | 后续是否被验证 |

### 三、交易规则库
提取作者反复提及或强调的交易纪律和方法论，每条规则包含：
- 规则内容（原文引用 + 你的理解）
- 出现频率（提到几次）
- 适用场景（什么市场环境下使用）
- 是通用规则还是特定情况规则

### 四、板块轮动规律
整理作者对各板块的观点演变：
1. 每个月份/阶段重点看好哪些板块
2. 板块切换的时间节点和触发因素
3. 板块内部的细分方向判断（如新能源→光伏vs锂电vs储能）
4. 哪些板块被持续看好/持续看空/中途转变看法
```
